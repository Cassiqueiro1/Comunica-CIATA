# Comunica-CIATA

Cliente Telegram acessível para iOS desenvolvido pelo CIATA (Centro de Inclusão Através da Tecnologia Assistiva).

## Objetivo

O Comunica-CIATA tem como objetivo oferecer uma experiência de uso do Telegram totalmente acessível para pessoas com deficiência visual, utilizando VoiceOver como tecnologia principal de interação.

O projeto não pretende ser apenas mais um cliente Telegram, mas sim um aplicativo cuja acessibilidade faça parte da arquitetura desde o primeiro dia de desenvolvimento.

## Princípios

* Acessibilidade em primeiro lugar.
* Interface simples.
* Navegação previsível.
* Alto desempenho.
* Baixo consumo de bateria.
* Privacidade.
* Código limpo.
* Documentação completa.

## Recursos planejados

* Login utilizando a TDLib.
* Lista de conversas acessível.
* Conversas organizadas por cabeçalhos de data.
* Navegação otimizada para VoiceOver.
* Ações rápidas utilizando o rotor do VoiceOver.
* Envio de mensagens.
* Resposta, encaminhamento, compartilhamento e exclusão.
* Reprodução de áudio.
* Visualização de documentos.
* Reconhecimento inteligente de imagens utilizando OpenAI ou Google Gemini.
* OCR.
* Configurações avançadas de acessibilidade.
